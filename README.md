> 业务跟进系统

* 1.普通业务员只能登录，放弃单，刷新单
* 2.超级管理员能 查看管理所有业务员、添加、删除、修改
* 3.业务员只能查看自己的单和无人单
* 4.业务员能抢单，然后就由他跟进
* 5.有操作日志，超级管理员能看到所有跟进，普通业务员只能看自己的跟进
* 6.超级管理员也有： 刷新单，强制放弃单等功能
* 7.当超级管理员将业务员删除时，需要将该业务员的所有单开放出来
* 8.完成单由业务员触发或者超级管理员触发，之前超级管理员点击关闭，本单就顺利关闭【完成关闭】。
* 9.普通业务员可以关闭没有完成的自己的单，然后由超级管理员确定后，该单就正式进入关闭中【失败关闭】。
* 10.只有超级管理员能将该单重新放出
